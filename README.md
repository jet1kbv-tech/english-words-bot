# English Words Bot

MVP Telegram-бота для совместного изучения английских слов двумя пользователями: Вовой (`@wp_bvv`) и Сашей (`@privetnormalno`). Бот работает как отдельный Python-проект и может быть запущен локально или как отдельный `systemd`-сервис на сервере.

## Возможности

- Доступ только для разрешённых Telegram username при первом входе.
- После регистрации пользователь хранится и определяется по `telegram_id`, поэтому смена username не ломает доступ.
- Добавление слов и фраз с переводом, опциональной темой и примером.
- Массовое добавление слов списком.
- Удаление слов из личного словаря.
- Личный словарь пользователя с пагинацией по 10 слов в одном сообщении.
- Режим `🔄 Обмен словами`: пользователь тренирует слова партнёра, а незнакомые слова добавляются в его личный словарь.
- Тренировка по своим словам отдельным режимом `🎯 Мои карточки`: smart review чаще поднимает новые и слабые слова, а после выбора `✅ Помню` / `❌ Не помню` бот автоматически показывает правильный ответ.
- Короткая игровая сессия `🎮 Игра на 10 слов`: бот заранее собирает до 10 слов из личного словаря без повторов, примерно 50% новых слов без `word_progress`, 35% слабых (`score <= 1` или забытых чаще, чем вспомненных) и 15% сильных (`score >= 2`), а при нехватке слов добирает из других категорий.
- Итог игровой сессии показывает ответы `✅ Знаю`, `❌ Не знаю`, `⏭ Пропущено`, сессии и карточки за сегодня, текущий streak и игровой уровень дня.
- Ежедневные напоминания через JobQueue в 14:00 `Europe/Moscow`: если сегодня нет завершённой игровой сессии, бот предлагает пройти 10 слов для продолжения streak.
- Отдельный прогресс для каждой пары `user_id` + `word_id`.
- SQLite-база создаётся автоматически при первом запуске.

## Локальный запуск

Требуется Python 3.11+.

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Заполните `.env` реальным токеном Telegram-бота:

```dotenv
BOT_TOKEN=123456:replace-with-real-token
DATABASE_PATH=english_words_bot.sqlite3
LOG_LEVEL=INFO
```

Запуск:

```bash
python main.py
```

## Переменные окружения

| Переменная | Описание | Значение по умолчанию |

## Seed words

Одноразовый импорт списка слов в личный словарь Саши (`@privetnormalno`) из раздела CHICAGA «Слова на английском языке для расширения словарного запаса» и следующих таблиц. Перед запуском Саша должен открыть `/start` в боте, чтобы пользователь появился в базе. Скрипт читает `DATABASE_PATH` из `.env` и пропускает уже добавленные слова.

```bash
python scripts/seed_sasha_chicaga_words.py
```

Старый импорт для Вовы (`@wp_bvv`) также доступен:

```bash
python scripts/seed_vova_words.py
```

## Запуск на сервере через systemd

Пример размещения проекта: `/opt/english-words-bot`.

```bash
cd /opt/english-words-bot
python3.11 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
nano .env
```

Пример unit-файла `/etc/systemd/system/english-words-bot.service`:

```ini
[Unit]
Description=English Words Telegram Bot
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
WorkingDirectory=/opt/english-words-bot
EnvironmentFile=/opt/english-words-bot/.env
ExecStart=/opt/english-words-bot/.venv/bin/python main.py
Restart=always
RestartSec=5
User=englishbot
Group=englishbot

[Install]
WantedBy=multi-user.target
```

Команды управления сервисом:

```bash
sudo systemctl daemon-reload
sudo systemctl enable --now english-words-bot
sudo systemctl status english-words-bot
journalctl -u english-words-bot -f
```

## Схема SQLite

Бот создаёт таблицы:

- `users`: `id`, `telegram_id`, `username`, `display_name`, `created_at`, `updated_at`.
- `words`: `id`, `owner_user_id`, `english`, `translation`, `topic`, `example`, `created_at`, `updated_at`.
- `word_progress`: `id`, `user_id`, `word_id`, `score`, `times_seen`, `times_remembered`, `times_forgotten`, `last_reviewed_at`, `created_at`, `updated_at`, уникальная пара `(user_id, word_id)`.
- `study_sessions`: завершённые и остановленные игровые сессии с количеством карточек, ответов `remembered` / `forgotten` / `skipped` и флагом `completed`.
- `daily_activity`: дневная активность пользователя с уникальной парой `(user_id, activity_date)`, количеством завершённых сессий и просмотренных карточек для streak.

## Команды и меню

- `/start` — регистрация/обновление пользователя и показ главного меню.
- `➕ Добавить слово` — пошаговое добавление слова.
- `📥 Добавить список слов` — массовое добавление слов из нескольких строк.
- `📚 Мой словарь` — только слова текущего пользователя, постранично по 10 слов; кнопки `⬅️ Назад` и `➡️ Далее` редактируют текущее сообщение.
- `🗑 Удалить` в личном словаре — показывает слова текущей страницы кнопками `🗑 1`, `🗑 2`, ... и удаляет только слова текущего пользователя после подтверждения.
- `🎯 Мои карточки` — тренировка по своим словам.
- `🎮 Игра на 10 слов` — короткая игровая сессия по личному словарю: после каждого ответа бот показывает правильный ответ и кнопку `➡️ Следующая карточка`, а после 10 карточек выводит итог дня и streak.
- `🔄 Обмен словами` — тренировка по словам партнёра: Вова видит слова Саши, Саша видит слова Вовы. Если нажать `❌ Не знаю`, бот сначала показывает правильный ответ, а затем копирует слово в личный словарь текущего пользователя без дублей по `owner_user_id + lower(english)` и сообщает статус добавления.
- `📊 Прогресс` — статистика текущего пользователя.

Карточки работают в двух направлениях и для каждой карточки направление выбирается случайно: English → Russian и Russian → English. После выбора `✅ Помню` / `❌ Не помню` или `✅ Знаю` / `❌ Не знаю` бот автоматически показывает правильный ответ и предлагает `➡️ Следующая карточка` или `🛑 Закончить`; кнопка `👀 Показать ответ` остаётся доступной для ручной проверки до выбора.

В игровой сессии доступны кнопки: `👀 Показать ответ`, `✅ Знаю`, `❌ Не знаю`, `⏭ Пропустить`, `🛑 Закончить`; после выбора ответа показывается правильный ответ и кнопка `➡️ Следующая карточка`.

Во время личной тренировки доступны кнопки: `👀 Показать ответ`, `✅ Помню`, `❌ Не помню`, `⏭ Пропустить`, `🛑 Закончить`. В режиме обмена доступны кнопки: `👀 Показать ответ`, `✅ Знаю`, `❌ Не знаю`, `⏭ Пропустить`, `🛑 Закончить`.
